什么是 Q-Learning？

📖 一句话定义

Q-Learning 是一种无模型（Model-Free）、离线策略（Off-Policy）的强化学习算法，通过维护一个 Q 表来记录每个状态下每个动作的价值，从而学习最优策略。

算法特点

🎯 无模型

不需要知道环境的转移概率和奖励函数，通过与环境交互直接学习。

🔀 离线策略

学习的策略与执行的动作策略可以不同，使用贪婪策略学习，ε-贪心策略执行。

⏱️ 时序差分

使用自举（Bootstrapping）方法，用当前估计更新当前估计。

应用场景

  • 🎮 游戏 AI（Atari、围棋等）
  • 🤖 机器人控制
  • 🚗 自动驾驶决策
  • 📊 资源调度优化
  • 🔍 推荐系统

核心概念

马尔可夫决策过程（MDP）

强化学习问题通常建模为 MDP，包含四个要素：

符号 含义 说明
S 状态空间（State） 环境所有可能的状态
A 动作空间（Action） 智能体可以执行的动作
P 转移概率（Transition） 执行动作后转移到新状态的概率
R 奖励函数（Reward） 执行动作后获得的即时奖励

Q 值（Quality Value）

Q 值表示在某一状态下执行某一动作的期望累积奖励：

Q(s, a) = 在状态 s 执行动作 a 后，能获得的期望总奖励

数学原理

贝尔曼最优方程

Q-Learning 的理论基础是贝尔曼最优方程：

Q*(s, a) = R(s, a) + γ · maxa' Q*(s', a')

Q 值更新公式

这是 Q-Learning 最核心的公式，必须牢记：

Q(s,a) ← Q(s,a) + α × [r + γ × maxaQ(s',a) - Q(s,a)]

参数详解

参数 名称 范围 作用
α 学习率 (0, 1] 控制新信息覆盖旧信息的程度
γ 折扣因子 [0, 1] 控制未来奖励的重要性
r 即时奖励 -∞ ~ +∞ 执行动作后获得的奖励
s' 下一状态 - 执行动作后的新状态

探索与利用

强化学习面临的核心问题：是选择当前已知的最佳动作，还是尝试新动作？

⚖️ ε-贪心策略

以 ε 的概率随机选择动作（探索），以 1-ε 的概率选择当前 Q 值最高的动作（利用）。

π(a|s) = ε/|A| (随机探索) 或 1-ε (选择最优)

算法流程

完整伪代码

Q-Learning Algorithm 
# 初始化
初始化 Q 表: Q(s, a) = 0, 对所有 s ∈ S, a ∈ A
设定学习率 α ∈ (0, 1], 折扣因子 γ ∈ [0, 1]
设定探索率 ε ∈ [0, 1], 衰减系数

# 主循环
for episode = 1, 2, ..., N:
    初始化状态 s₁
    
    for t = 1, 2, ..., T:
        # 1. 选择动作 (ε-贪心策略)
        if random() < ε:
            aₜ ← 随机选择动作           # 探索
        else:
            aₜ ← argmaxₐ Q(sₜ, a)     # 利用
        
        # 2. 执行动作，观察结果
        执行动作 aₜ, 观察奖励 rₜ₊₁ 和下一状态 sₜ₊₁
        
        # 3. Q 值更新 (核心!)
        Q(sₜ, aₜ) ← Q(sₜ, aₜ) + α[rₜ₊₁ + γ·maxₐ Q(sₜ₊₁, a) - Q(sₜ, aₜ)]
        
        # 4. 状态转移
        sₜ ← sₜ₊₁
        
        # 5. 检查终止条件
        if sₜ 是终止状态:
            break
    
    # 6. 衰减探索率
    ε ← max(ε × decay, ε_min)

返回 Q 表

算法步骤详解

  1. 初始化

    创建 Q 表，初始值设为 0。设定学习率 α、折扣因子 γ、探索率 ε。

  2. 开始新回合

    重置环境到初始状态，准备开始一轮新的探索。

  3. 选择动作

    使用 ε-贪心策略：以 ε 概率随机探索，以 1-ε 概率选择当前最优动作。

  4. 执行并观察

    执行选定的动作，观察环境返回的奖励和新状态。

  5. 更新 Q 值

    使用 TD 误差更新 Q 表：向 TD 目标移动一小步（步长为 α）。

  6. 重复直到结束

    循环执行直到到达终点、踩中陷阱或超过最大步数。

  7. 衰减探索率

    逐渐减小 ε，从探索转向利用。

代码实现

环境类

grid_world_env.py 
import numpy as np

class GridWorldEnv:
    """网格世界环境"""
    
    def __init__(self, size=4):
        self.size = size
        self.start_pos = (0, 0)
        self.goal_pos = (size-1, size-1)
        self.traps = [(1, 1), (2, 2)]
        self.action_space = 4  # 上、下、左、右
        
    def reset(self):
        """重置环境"""
        self.current_pos = self.start_pos
        self.steps = 0
        return self._get_state()
    
    def step(self, action):
        """执行动作"""
        row, col = self.current_pos
        
        # 移动
        if action == 0: row = max(0, row-1)      # 上
        elif action == 1: row = min(self.size-1, row+1)  # 下
        elif action == 2: col = max(0, col-1)    # 左
        elif action == 3: col = min(self.size-1, col+1)  # 右
        
        self.current_pos = (row, col)
        
        # 计算奖励
        reward = -1
        done = False
        
        if self.current_pos == self.goal_pos:
            reward = 10
            done = True
        elif self.current_pos in self.traps:
            reward = -10
            done = True
        elif self.steps >= 100:
            done = True
            
        return self._get_state(), reward, done, {}
    
    def _get_state(self):
        """获取状态索引"""
        return self.current_pos[0] * self.size + self.current_pos[1]

智能体类

q_learning_agent.py 
import numpy as np

class QLearningAgent:
    """Q-Learning 智能体"""
    
    def __init__(self, n_states, n_actions, lr=0.1, gamma=0.99):
        self.n_states = n_states
        self.n_actions = n_actions
        self.lr = lr          # 学习率 α
        self.gamma = gamma    # 折扣因子
        self.epsilon = 1.0    # 探索率
        
        # 初始化 Q 表
        self.q_table = np.zeros((n_states, n_actions))
    
    def choose_action(self, state, training=True):
        """ε-贪心策略选择动作"""
        if training and np.random.random() < self.epsilon:
            return np.random.randint(self.n_actions)  # 探索
        else:
            return np.argmax(self.q_table[state])     # 利用
    
    def learn(self, state, action, reward, next_state, done):
        """Q-Learning 更新"""
        current_q = self.q_table[state, action]
        
        # 计算 TD 目标
        if done:
            target = reward
        else:
            target = reward + self.gamma * np.max(self.q_table[next_state])
        
        # 更新 Q 值
        self.q_table[state, action] += self.lr * (target - current_q)
    
    def decay_epsilon(self, decay=0.995, min_val=0.01):
        """衰减探索率"""
        self.epsilon = max(min_val, self.epsilon * decay)

训练脚本

train.py 
from grid_world_env import GridWorldEnv
from q_learning_agent import QLearningAgent

# 创建环境和智能体
env = GridWorldEnv(size=4)
agent = QLearningAgent(n_states=16, n_actions=4)

# 训练
for episode in range(1000):
    state = env.reset()
    total_reward = 0
    done = False
    
    while not done:
        # 选择动作
        action = agent.choose_action(state)
        
        # 执行动作
        next_state, reward, done, _ = env.step(action)
        
        # 学习
        agent.learn(state, action, reward, next_state, done)
        
        state = next_state
        total_reward += reward
    
    # 衰减探索率
    agent.decay_epsilon()
    
    if (episode + 1) % 100 == 0:
        print(f"Episode {episode+1}, Reward: {total_reward:.2f}, "
              f"Epsilon: {agent.epsilon:.3f}")

print("训练完成！")

实践技巧

参数调优

参数 太小 合适 太大
学习率 α 学习极慢 0.1 ~ 0.3 不稳定震荡
折扣因子 γ 短视只看眼前 0.9 ~ 0.99 可能发散
探索率 ε 陷入局部最优 1.0→0.01 学习不稳定

常见问题

Q1: Q 值不收敛怎么办？
  • 尝试减小学习率到 0.05 或 0.01
  • 增加训练轮数
  • 检查奖励设置是否合理
Q2: 智能体总是走同一个路径？
  • 检查 ε 是否衰减太快
  • 确保探索率不会降到 0
  • 尝试更长的训练时间
Q3: 踩陷阱次数太多？
  • 增加陷阱的惩罚值
  • 延长探索阶段的 ε 值
  • 添加更多训练轮数
